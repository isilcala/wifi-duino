wifiDuino
==========

wifiDuino是WiFi-Duino的库函数，文档和使用教程以及WiFi-Duino的更多信息请参考：[Wifi-Duino](http://winterland.duapp.com/wifi-duino/)

wifiDuino功能简介：

+ 简单方便的建立热点或者连接到其他路由器上。
+ 简单方便的建立Http服务器或者发送Http请求。
+ 支持socket连接，可以实现服务器push
